# Marketing Campaign Performance Analysis

## Overview
This project analyzes marketing campaign data to track key performance indicators (KPIs) and optimize campaign performance.

## Tools Used
- SQL
- Python (Pandas)
- Power BI

## Key Metrics
- Click-Through Rate (CTR)
- Conversion Rate
- Cost per Conversion

## Key Insights
- Compared campaign performance based on conversions
- Identified trends and anomalies in campaign data
- Built dashboard for real-time KPI monitoring

## Project Structure
- data/ → dataset
- sql/ → SQL queries
- python/ → analysis script
